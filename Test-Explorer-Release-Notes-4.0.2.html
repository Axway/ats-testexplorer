ATS : Test Explorer Release Notes 4.0.2

15 November 2017

New Features

 Test Explorer short video presentation

You can watch it here


Support PostgreSQL for log database

So far the only choice for Test Explorer database was to run on MS SQL Server. Now PostgreSQL is another option which allows you using a non Windows host.

Here is the Test Explorer install guide

Here is more about ATS Logging System


 Multiple charts for system statistics

You can now combine your statistics in more than one chart. See the "Testcase statistics tab" section here


 The scenario description can be displayed and edited in separate modal dialog


 Show time stamps according to the browser usage time zone

Every event persisted in the log database is now using UTC time, so it does not matter the time zone you log from. For example, you might have the test executed in one time zone while some operations on ATS Agent executed in another time zone.

While reviewing, the timestamps are fixed by applying the time zone where the web browser is running.
